# MedCare Clinic & Pharmacy — Excel Mini Project
### Route Upskill Academy | Data Analysis Program

## About This Project
A mini data analysis project using real-looking clinic and pharmacy 
records. You will explore 320 patient visit records, derive meaningful 
columns, and answer business questions using only Excel formulas.

## Dataset
- File: `MedCare_Clinic_MiniProject.xlsx`
- Records: 320 patient visits
- Period: 2023–2024
- Domain: Clinic + Pharmacy

## Tools Used
- Microsoft Excel (formulas only)

## Project Structure
The Excel file contains the following sheets:
| Sheet | Purpose |
|-------| --------|
| Raw Data | Original dataset — do not edit |
| Working Sheet |Create Your cleaned data and derived columns |  
| Changelog | Document every change you make |
| Summary Report | Your formulas, answers, and written findings |
| Question Set (A/B/C/D/E) | Your assigned questions |

## How to Submit
1. Fork this repository
2. Download the Excel file
3. Complete your analysis
4. Update this README with your Problem, Methodology, findings, etc 
5. Upload your completed Excel file to your forked repo
6. Share your fork link with your trainer
